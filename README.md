# 💻bunlog
## 🚀 소개
### 프로젝트 소개
내가 내 손으로 만든 블로그 사이트 하나 가져보고 싶어서 만들었습니다.

### 주요 기능
- MDX로 글 작성 가능
- 깔끔한 디자인
- 유지보수 용이

### 기술적 특징
- Next.js : 풀스택 가능 & MDX 지원
- shadcn/ui : 깔끔한 UI 제공

## 💻 설치 방법
1. 저장소 복제
```bash
git clone [레포지토리 URL]
cd [프로젝트명]
```

2. 의존성 설치
```bash
npm install
# 또는
bun install
```

3. 개발 서버 실행
```bash
npm run dev
# 또는
bun dev
```

4. 빌드
```bash
npm run build
# 또는
bun build
```
